We can do the same thing using %>%. This function as has no builtin meaning but the user (or a package) can define operators of the form %whatever% in any way they like. For example, this function will return a string consisting of its left argument followed by a comma and space and then its right argument.

Monthly <- SCZ %>%
  mutate(Date = ymd(Date),Year = year(Date), Month = month(Date)) %>%
  group_by(Month) %>%
  summarise(mean = mean(Daily.Mean.PM2.5.Concentration), 
            sd = sd(Daily.Mean.PM2.5.Concentration), 
            N = length(Daily.Mean.PM2.5.Concentration) )

Estimating Confidence Intervals

Although the standard deviation describe variability, the statistic is not explicilty associated with a probability. We cna convert these to probabilities by mulitplying by a theoretical probability distribution.

qnorm(.975)
## [1] 1.959964
Monthly$UCL95 = Monthly$mean + qnorm(.975)*Monthly$sd/sqrt(Monthly$N)
Monthly$LCL95 = Monthly$mean - qnorm(.975)*Monthly$sd/sqrt(Monthly$N)

If we obtain a mean concentration of 14 using our sensors, we can easily test the value using t.test and all the Septmber values. So, well create a vector of all the September readings.
